# Flavor Mapping

A nerdy way to visualize allergens and the chemical composition of food. 🔬

Flavor Mapping goes beyond simple tracking; it integrates advanced flavor matching features to help you discover perfect pairings and creative recipes based on chemical profiles.

## Features

- **Chemical Visualization**: Deep dive into the molecular components of your favorite ingredients.
- **Allergen Tracking**: Visualize allergen profiles across different food groups.
- **Flavor Pairing**: Smarter pairing suggestions based on real chemical compatibility.
- **Recipe Discovery**: Find recipes that match your flavor maps.

## Tech Stack

This project is built with [Next.js](https://nextjs.org), optimized for performance and type-safe development.
